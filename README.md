# flxstore-site

Loja virtual desenvolvida com **Next.js**, focada em produtos de academia, esportes, eletrônicos, celulares e muito mais.

---

## 🚀 Tecnologias utilizadas
- **Next.js 14**
- **React 18**
- Hospedagem automática na **Vercel**
- Controle de versão no **GitHub**
